# Working with AI

AI assistants can help you build with Nimiq by accessing documentation, RPC methods, and developer resources through standardized protocols.

## Available Integrations

### [Model Context Protocol (MCP)](/working-with-ai/mcp)

Enables AI assistants to access structured Nimiq documentation, RPC methods, and blockchain data through standardized protocol.

**Best for**: Deep integration with Claude Code, ChatGPT, Cursor, VS Code Copilot, Windsurf, Zed

**Server**: `https://nimiq.com/developers/mcp`

**Features**:
- Documentation discovery with module filtering
- Guided prompts for finding relevant docs
- Interactive access to Nimiq resources

[Learn about MCP →](/working-with-ai/mcp)

### [LLMs.txt](/working-with-ai/llms-txt)

Structured documentation optimized for large language models through `/llms.txt` endpoints.

**Best for**: General AI assistance in any chat interface

**Endpoints**:
- `https://nimiq.com/developers/llms.txt` (~5-10K tokens)
- `https://nimiq.com/developers/llms-full.txt` (~200K+ tokens)

**Features**:
- Simple text-based doc access
- Works with any AI that supports web fetching
- No setup required

[Learn about LLMs.txt →](/working-with-ai/llms-txt)

## Why Use AI with Nimiq?

**Faster Development** - Instant answers about protocol details, RPC methods, integration patterns

**Code Generation** - Generate Nimiq transactions, queries, blockchain interactions

**Documentation Search** - Find relevant guides and API references quickly

**Best Practices** - Learn recommended patterns for building with Nimiq

## Getting Started

Choose approach that works best for your workflow:

- **MCP** - Deep integration with AI assistants (Claude Code, ChatGPT, Cursor, etc.)
- **LLMs.txt** - General AI assistance with Nimiq docs in any chat interface

Both approaches kept in sync with latest Nimiq documentation.
